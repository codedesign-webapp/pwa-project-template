# PWA-Camera 프로젝트 템플릿

본 소스는 'Do it! 프로그레시브 웹앱 만들기' 책의 예제를 토대로해서 '카메라' PWA 앱의 프로젝트 시 활용할 수 있는 보일러플레이스 소스입니다.

## 추가된 기능
```
* 전면/후면 카메라 전환
```
