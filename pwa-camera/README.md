# PWA-Camera 프로젝트 템플릿

본 소스는 `'Do it! 프로그레시브 웹앱 만들기' 책`의 예제를 토대로해서 `'카메라 PWA' 앱`의 프로젝트 시작시 활용할 수 있는` 보일러플레이스 소스`입니다. 

## 기능
```
1. 카메라로 사진 촬영
2. 파이어베이스 DB, 스토리지 업로드 저장
3. 촬영한 사진 갤러리 조회, 수정, 삭제
4. 전면/후면 카메라 전환
```
