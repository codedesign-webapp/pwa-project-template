# PWA-Login-popup-redirect 프로젝트 템플릿

본 소스는 `'Do it! 프로그레시브 웹앱 만들기' 책`의 예제를 토대로해서 `'구글 로그인 앱`의 프로젝트 시작시 활용할 수 있는` 보일러플레이스 소스`입니다. 

## 기능
```
1. 구글 로그인 oAuth 구현
2. 파이어베이스 구글 로그인 연동
3. 구글 계정 팝업 선택 적용
4. 구글 계정 리다이렉트 선택 적용
5. 반응형 웹으로 로그인 메뉴 실시간 변경 구현
6. 로그인 기능과 뷰 라우터 연동
```
